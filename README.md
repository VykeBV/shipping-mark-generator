# Vyke Create — Shipping Mark Generator

A static, browser-based authoring tool for print-ready shipping mark / ex-mark / export-carton labels.

Sister project to the Vyke Create Data Sheet generator — same dark Tweaks-panel UI and live-preview workflow, but tailored to producing rectangular shipping marks with vector-precise EAN-13 barcodes, ISO 780 handling icons, configurable text rows, customer logo, bleed and crop marks.

## Run locally

```sh
npm install
npm run dev
```

Then open http://localhost:4321/shipping-mark-generator/

## Build

```sh
npm run build
```

The static site is emitted to `dist/`.
